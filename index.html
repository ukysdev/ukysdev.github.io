Fast. Minimal. Sharp.

Viper Console ist eine kompakte Entwickler-Konsole mit Shell-Passthrough, Aliases, Jobs (pm2-light), Mini-Servern (static / node / python), Profilen und Session-Env Overrides — gebaut für schnelle Workflows.

Aliases & Commands

Registriere eigene Commands (addcmd) inkl. Default-Args und cwd für maximale Produktivität.

Built-in Server

Static server built-in, plus node http-server oder python http.server — instant serving.

Commands Overview

Core & Navigation

  • help, exit, cls/clear, history
  • pwd, cd, ls/dir, which, whichall
  • tree, cat, less, rm, cp, mv, stat
  • run, cmd (Windows), sh (bash)

Jobs & Serving

  • start, jobs, stop, kill, restart, logs
  • serve, serve-node, serve-python
  • addcmd, alias, rmcmd, unalias
  • profile add/use/list, env, set

Workflows & Examples

Profiles

# Create profile
profile add dev --cwd . NODE_ENV=development

# Use it
profile use dev

# List
profile list

Jobs

# Start process
start web --restart always node server.js

# Monitor
jobs
logs 1 --tail 200

# Control
stop 1 / restart 1
